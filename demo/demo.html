⬡ LazyHTML the attribute is the stylesheet

Combine everything on the tag itself.

No stylesheet round-trip, no class-naming debates. LazyHTML reads attributes off the element — color, spacing, layout, hover states, breakpoints — and resolves them lazily into one shared CSS class.

1. Typography & color

Bold, underlined, italic, and colored — four attributes, one element.

color="black" — named colors resolve normally.

color="#ff0044" — proper hex works as expected.

color="#black" — the typo-tolerant form from the spec still resolves.

2. Spacing & sizing
